# parse_yaml

Parses a string as a YAML map.

```tql
parse_yaml(input:string, [schema=string, selector=string, schema_only=bool,
           raw=bool, unflatten=string]) -> record
```

## Description

The `parse_yaml` function parses a string as a YAML map.

### `input: string`

The string to parse.

<CommonOptions />

## Examples

### Parse a YAML map

```tql
from { x: r#"yarp:
  - darp
  - larp"# }
x = x.parse_yaml()
```
```tql
{
  x: {
    yarp: [
      "darp",
      "larp",
    ],
  },
}
```
