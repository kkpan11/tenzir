# save_ftp

Saves a byte stream via FTP.

```tql
save_ftp url:str [
load_tcp endpoint:string, [parallel=int, tls=bool, cacert=string, certifle=string,
                           keyfile=string, skip_peer_verification=bool,
                           skip_host_verification=bool]
```

## Description

Saves a byte stream via FTP.

### `url: str`

The URL to request from. The `ftp://` scheme can be omitted.

<TLSOptions tls_default="true"/>

## Examples

```tql
save_ftp "ftp.example.org"
```
