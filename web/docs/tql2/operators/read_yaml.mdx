# read_yaml

Parses an incoming YAML stream into events.

```tql
read_yaml [merge=bool, raw=bool, schema=string, selector=string, schema_only=bool, unflatten=string]
```

## Description

Parses an incoming [YAML](https://en.wikipedia.org/wiki/YAML) stream into events.

<CommonOptions />

## Examples

### Parse a YAML file

```yaml title="input.yaml"
---
name: yaml
version: bundled
kind: builtin
types:
  - parser
  - printer
dependencies:
  []
...
```

```tql
load_file "input.yaml"
read_yaml
```

```tql
{
  name: "yaml",
  version: "bundled",
  kind: "builtin",

    "parser",

  ],

```
